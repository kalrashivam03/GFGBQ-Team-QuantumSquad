TruthbyAi

Ensuring Trust in AI Generated Information.

About This:

  1. TruthbyAI verifies AI responses by checking them with reliable sources and highlights whether the information is Authentic, Partially True, or Incorrect.
  2. Working - In this system, user enters the text or an image then our system checks the information by comparing it with trusted databases and fact-checking resources then displays the result. So by using this application we can ensure the accuracy of AI-generated information.
  3. We had used Java for backend development,HTML and CSS for frontend development and GitHub for version control.
