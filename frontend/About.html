TruthbyAI

Ensuring Trust in AI Generated Information.

About This

  1. TruthbyAI verifies AI responses by checking them with reliable sources and highlights whether the information is Authentic, Partially True, or Incorrect.
  2. Working: Users can input text or images. The system compares the content with trusted databases and fact-checking resources to ensure the accuracy of AI-generated information.
  3. Technologies used include Java for backend, HTML & CSS for frontend, and GitHub for version control.
